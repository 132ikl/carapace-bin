diff
Diff live version against would-be applied version
--field-manager VAL  Name of the manager used to track field ownership.
-f, --filename VAL  Filename, directory, or URL to files contains the configuration to diff
--force-conflicts   If true, server-side apply will force the changes against conflicts.
-k, --kustomize VAL  Process the kustomization directory. This flag can't be used together with -f or -R.
-R, --recursive   Process the directory used in -f, --filename recursively. Useful when you want to manage related man
-l, --selector VAL  Selector (label query) to filter on, supports '=', '==', and '!='.(e.g. -l key1=value1,key2=value2)
--server-side   If true, apply runs in the server instead of the client.
