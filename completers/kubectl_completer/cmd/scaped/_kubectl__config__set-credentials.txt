_kubectl__config__set-credentials
set-credentials
Sets a user entry in kubeconfig
--auth-provider VAL  Auth provider for the user entry in kubeconfig
--auth-provider-arg VAL  'key=value' arguments for the auth provider
--embed-certs VAL  Embed client cert/key for the user entry in kubeconfig
--exec-api-version VAL  API version of the exec credential plugin for the user entry in kubeconfig
--exec-arg VAL  New arguments for the exec credential plugin command for the user entry in kubeconfig
--exec-command VAL  Command for the exec credential plugin for the user entry in kubeconfig
--exec-env VAL  'key=value' environment values for the exec credential plugin
